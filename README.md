# سیستم مدیریت دانشجویان (Student Management System)

یک برنامه کنسولی برای مدیریت دانشجویان، درس‌ها و نمرات به زبان C#

## ویژگی‌ها

✅ **مدیریت دانشجویان**
- ثبت دانشجوی جدید
- مشاهده تمام دانشجویان
- جستجوی دانشجو
- حذف دانشجو

✅ **مدیریت درس‌ها**
- اضافه کردن درس جدید
- ثبت نمره دانشجو

✅ **رابط کاربری فارسی**

## نحوه کار کردن

### الزامات
- .NET 6.0 یا بالاتر

### کامپایل و اجرا

```bash
# کامپایل کردن
dotnet build

# اجرا کردن
dotnet run
```

## ساختار فایل‌ها

- **Program.cs** - برنامه اصلی و رابط کاربری
- **Student.cs** - کلاس دانشجو
- **Course.cs** - کلاس درس
- **StudentCourse.cs** - کلاس ثبت‌نام دانشجو در درس
- **Database.cs** - مدیریت داده‌ها
- **StudentManagementSystem.csproj** - فایل پروژه
